Sistema Operacional

Sistema operacional: É um assunto bastante interessante mais um pouco complexo, fala sobre como o sistema operacional está atuando nos computadores, celulares, tablets etc... O SO é um software com um conjunto de funções que irão servir como intermediarias para fazer a comunicação entre hardware e software do meio tecnológico parar que o usuário tenha de forma mais fácil aquilo que ele deseja está utilizando em seu aparelho.

Dentre as funções básicas de computadores de uso geral, pode-se citar:


• definição da interface com o usuário;
• compartilhamento de hardware entre usuários;
• compartilhamento de dados entre usuários;
• gerenciamento dos dispositivos de entrada e saída;
• tratamento e recuperação de erros

O tema explica também como ocorre a troca de dados por meio dos barramentos da placa-mãe para que não tenha conflito e perca de arquivos e também como o sistema está a todo momento gerenciando e organizando a troca de informações enquanto o computador está ligado para que mantenha o sistema eficiente e seguro para o usuário.
